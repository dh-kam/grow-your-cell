기능 가이드

• WASD : 관성 가속도 제어 (정지 없이 계속 표류)

• SPACE : 세포 분열 (Mitosis) (반절 질량 방출 및 질주)

• SHIFT : 미토콘드리아 대시 (질량 4% 소모하여 질주)

• Q : 리소좀 산성 투사체 (마우스 방향 발사, 적 세포 수축)

• E : 포자화 (Sporulation) (이동 불가 상태 무적 실드)

• CTRL : 섬모 제동 (Cilia Brake) (마찰 급증, 정지/코너링)

• WHEEL : 현미경 배율 조정 (화면 확대/축소)

• M : AI 본능 조작 전환

• TAB / 세포 클릭 : 분열 세포 포커스 전환

• R : 포커스 세포 재결합

• 보라색 암세포 : 접촉한 세포를 파괴하거나 암세포화

• HF1 : 이 패널 표시/숨김
